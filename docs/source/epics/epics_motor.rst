
==============
Motors control
==============

.. module:: py4syn.epics.MotorClass
   :synopsis: Python class using basic Motor Record fields.

This is the basic Python class to control motor devices by using EPICS set of
tools to provide the minimal infrastructure for use in managing and operating
motor devices by reading and/or setting their record fields - also known as PVs,
`Process Variables`.

You will find below a set of methods which is available at this version of
the Motor's class with their expected input parameters and/or waited results
(if any), describing involved data types.
