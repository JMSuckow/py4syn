
===========================
Hyppie CCD Detector control
===========================

.. module:: py4syn.epics.HyppieCCDClass
   :synopsis: Python Class for EPICS LabView RT Detector control module

This is the basic Python class to help in the configuration and control of CCD
devices by using EPICS set of tools to provide the minimal infrastructure for
use in managing and operating motor devices by reading and/or setting their
record fields - also known as PVs, `Process Variables`.

Charge-Coupled Device (CCD) is a silicon light sensor which converts
incident light, or photons, into electrons.
