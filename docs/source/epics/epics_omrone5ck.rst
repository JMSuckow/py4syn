
=================================
Omron E5CK Temperature Controller
=================================

.. module:: py4syn.epics.OmronE5CKClass
   :synopsis: Python class for Omron E5CK temperature controllers

The Omron E5CK temperature controllers are programmable temperature controllers. The
Py4Syn implementation is strongly software based, overcoming device programming
limitations, allowing it to be used like an IScannable device and supporting tens of
thousands of ramp steps.
