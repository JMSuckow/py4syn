Welcome to Py4Syn
=================

|docs|  |health|

Purpose
-------

Py4Syn is a combined effort to provide a minimum operational set of Python
classes and scripts which support the basic and common operations at the Brazilian
Synchrotron Light Laboratory (LNLS).
